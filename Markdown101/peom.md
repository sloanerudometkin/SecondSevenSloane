# Markdown 101
## Paragraph about markdown:
Markdown lets you add structure and styling (like headings and bolding) without having to use the complex HTML tags. It is written in plain text, so Markdown files are able to be opened/read in any text editor: so it is pretty portable. When you are ready to display the styled text, for example on a website or a document, a Markdown processor is used to convert the simple text to the formatted HTML/PDF. It doesn't have to be rendered, it is perfectly easy to read as plain text- some common formatting procedures are #'s for headings, 1 "*" or "_" for italicizing, and "-" for a list.
